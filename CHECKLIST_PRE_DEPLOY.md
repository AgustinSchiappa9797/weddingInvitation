# Checklist final de deploy

## Antes de publicar
- Confirmar que el link usa el token correcto para cada invitado.
- Verificar que las fotos definitivas cargan bien en desktop y mobile.
- Confirmar que el fondo principal se vea al abrir la invitación.
- Revisar que el monograma diga `N & A`.
- Validar que la sección Detalles no repita dirección.
- Confirmar que la sección Dirección tenga el link o mapa correcto.
- Reemplazar `assets/audio/boda.ogg` por el audio final.
- Probar música en Chrome desktop, Safari iPhone y Android Chrome.
- Confirmar que la playlist abra bien y que el embed se vea si el link es de Spotify.
- Probar la galería y el lightbox en mobile.
- Validar la confirmación con un envío real.
- Confirmar texto de éxito y error en Confirmación.
- Revisar que el botón sticky de Confirmación no tape contenido en mobile.
- Verificar que el botón de música no se pise con el sticky.
- Revisar que el gato animado se vea y recorra la pantalla.
- Probar la invitación con conexión lenta para validar loading y fondos.

## Antes del deploy final
- Hacer una última prueba en ventana privada.
- Limpiar caché o versionar assets si el hosting cachea estáticos.
- Confirmar metadatos para compartir si vas a pasar el link por WhatsApp.
- Guardar una copia del zip final de deploy.
